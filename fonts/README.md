# Tipografías auto-hosteadas

Archivo (subset variable, pesos 400–600) e IBM Plex Mono (400, 500), subconjuntos `latin` + `latin-ext`. Vendorizadas desde Google Fonts para no depender de `fonts.googleapis.com`/`fonts.gstatic.com` en producción (§11 de la especificación: sin recursos externos).

- Archivo — Copyright 2020 The Archivo Project Authors
- IBM Plex Mono — Copyright © 2017 IBM Corp. con Reserved Font Name "Plex"

Ambas bajo SIL Open Font License 1.1 (`OFL.txt`).

Cuando se agreguen idiomas con otros scripts (cirílico para ruso, CJK para japonés/coreano/chino — v1.1, §4.4), habrá que sumar los subconjuntos correspondientes o una tipografía de respaldo con esa cobertura.
